# ShadowSense Extension

## What It Does

- Shows current live-user and alert counts in popup
- Pulls recent alerts from backend APIs
- Displays alert count badge on extension icon

## Setup

1. Open browser extension management page
2. Enable Developer Mode
3. Click Load unpacked
4. Select this `extension` folder

## Notes

- Backend must run at `http://localhost:5000`
- Update API URL in `popup.js` and `background.js` for production
